# Library Inventory Manager

A small command-line library inventory manager demonstrating object-oriented design,
JSON persistence, exception handling, and a basic CLI.

## Features
- Add, issue, return books
- Search by title or ISBN
- Persist data into `books.json`
- Logging and exception handling
- Simple unit tests

## Run
```bash
python -m cli.main
```

## Project structure
See `library_manager/` for core modules and `cli/main.py` for the interface.
